◆ finish()

template<typename ... WriteArgs>
void helios::filems::MultiLasSyncFileWriter< WriteArgs >::finish ( )
inlineoverridevirtual

MultiLasSyncFileWriter updates each header and guarantees all writings have been done only after the finish method has been invoked. If it has not been manually invoked, then it will when destroying the instance. Once the finish method has been invoked, the MultiLasSyncFileWriter should not be used again.
