◆ setSpecularity()

void Material::setSpecularity ( )

Compute the specularity \(s\).

Let \(Kd\) represent the diffuse components and \(Ks\) represent the specular components

\[ s = \frac{\sum_{i=1}^{3}{Ks_{i}}} {\sum_{i=1}^{3}{Kd_{i}} + \sum_{i=1}^{3}{Ks_{i}}} \]
